# KİŞİSEL WEB SİTE
Projemiz Kişisel bir portfolio sitesidir. Tek katmnalı mimari üzerine asp.net core 6.0 ile geliştirilmiştir.

## Kullanılan Teknolojiler
- Asp.Net Core 6.0
- Admin Panelli
- Tek Katmanlı
- JavaScript
- Bootstrap
- Identity
- MSSQL
- HTML
